> Attributes

参数 | 说明 | 类型 | 可选值 | 默认值
---|---|---|---|---
bottom | 组件距离底部的距离 | number | - | 30
right | 组件距离右部的距离 | number | - | 30
height | 页面滚动高度达到该值时才显示BackTop组件 | number | - | 400
className | 给组件添加一个类 | string | - | -
